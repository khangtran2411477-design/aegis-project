# Aegis Project

## Setup
1. Tạo môi trường ảo:
   `python -m venv venv`

2. Kích hoạt môi trường ảo:
   - Windows (CMD): `venv\Scripts\activate`
   - Windows (Git Bash): `source venv/Scripts/activate`
   - Linux/Mac: `source venv/bin/activate`

3. Cài thư viện:
   `pip install -r requirements.txt`

4. Chạy script mẫu:
   `python hello_aegis.py`
